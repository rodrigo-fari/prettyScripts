    bash

sudo apt install cmatrix
cmatrix -C green -s

(Press Q to exit.)

Transparent background (add to kitty.conf):
conf

background_opacity    0.85
dynamic_background_opacity yes

Green-themed shell prompt (e.g., for Bash/Zsh):
bash

echo 'export PS1="\[\e[32m\][\u@\h \W]\$\[\e[0m\] "' >> ~/.bashrc
source ~/.bashrc


Why This Works

    #00FF41 is the iconic "Matrix green" (HEX color used in the movie).

    Block cursor + monospace font mimics the code aesthetic.

    Powerline tabs add a modern touch while keeping the theme cohesive.

For the full "I know kung fu" experience, pair this with:

    A tiling WM (like i3 or AwesomeWM).

    Neofetch in green:
    bash

neofetch --ascii_colors 2 10


3. Falling Code Effect

Create a shortcut in ~/.bashrc or ~/.zshrc:
bash

echo 'alias matrix="cmatrix -C green -s"' >> ~/.bashrc
source ~/.bashrc

Run with:
bash

matrix  # Press 'Q' to exit

4. Matrix-Themed Shell Prompt

Add to your shell config (~/.bashrc or ~/.zshrc):
bash

echo 'export PS1="\[\e[32m\]>\[\e[1;32m\]>\[\e[0;32m\]> \[\e[0m\]"' >> ~/.bashrc
source ~/.bashrc

5. Startup Command for Full Effect

Add this to ~/.config/kitty/kitty.conf to auto-run effects:
conf

shell_integration enabled
initial_window_width    120c
initial_window_height   30c
enabled_layouts         tall:bias=50

1. Local Kitty Installation
bash

# Download and extract Kitty to ~/.local/
mkdir -p ~/.local/{bin,share}
curl -L https://github.com/kovidgoyal/kitty/releases/download/v0.30.1/kitty-0.30.1-x86_64.txz | tar -xJ -C ~/.local/share/
ln -sf ~/.local/share/kitty/bin/kitty ~/.local/bin/

2. Matrix Configuration (No Root Needed)

Create ~/.config/kitty/kitty.conf:
conf

# === MATRIX MODE ===
foreground      #00FF41
background      #000000
cursor          #00FF41
cursor_shape    block
font_size       14

# Use built-in terminal font (no root needed)
font_family     monospace

# Colors
color0          #000000
color8          #0A280A
color2          #00FF41
color10         #00FF41

# Disable decorations
hide_window_decorations yes

3. Local Cmatrix Installation
bash

# Compile cmatrix locally
mkdir -p ~/src && cd ~/src
git clone https://github.com/abishekvashok/cmatrix.git
cd cmatrix
./configure --prefix=$HOME/.local
make && make install

4. Matrix Aliases (Add to ~/.bashrc)
bash

echo '
# Matrix mode
alias matrix="$HOME/.local/bin/cmatrix -C green -s"
export PATH="$HOME/.local/bin:$PATH"
' >> ~/.bashrc
source ~/.bashrc

5. Run Without Root

    Start Kitty:
    bash

~/.local/bin/kitty

Launch Matrix mode:
bash

    matrix

    (Press Q to exit)
